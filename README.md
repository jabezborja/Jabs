## Shorten a Link
To shorten a link, just go the [website](https://jabsga.herokuapp.com/zGGGAuV) then paste the link you want to be shorten and Jabs.ga will do the job for you. You can edit the title of the link, customize the back-half of the link, the choice is all yours, it is fully customizable and it's free ðŸ‘Œ
