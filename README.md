# Shortening a link made easy for absolutely everyone.

Created with:
- [Python Django Web Framework](https://djangoproject.com)
- [Tailwind CSS](https://tailwindcss.com)

As of now, the URL of Jabsga which is `jabs.ga/` is not yet available because Heroku disabling it without a card, but soon it will be short as it should be.

## Shorten a Link
To shorten a link, just go the [website](https://jabsga.herokuapp.com/) then paste the link you want to be shorten and Jabs.ga will do the job for you. You can edit the title of the link, customize the back-half of the link, the choice is all yours, it is fully customizable and it's free ðŸ‘Œ
